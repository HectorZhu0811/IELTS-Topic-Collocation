## Fields

The files are UTF-8 tab-separated files with HTML enabled.

Fields:

1. Front
2. Back
3. Topic
4. TopicZh
5. Type
6. BaseChinese
7. BaseEnglish
8. HighlightChinese
9. HighlightEnglish
10. Tags
11. CardId

In Anki, map `Front` and `Back` to your card template fields. You can map `Tags` as the tag field if your import screen offers that option.
