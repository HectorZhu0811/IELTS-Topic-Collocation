Zen transition previews

把掌握情况变成下一张卡片的过渡。

三种方案都从同一个动作开始：英文面出现后，学生再次按空格或点击掌握按钮，先看到掌握反馈，再进入下一张。

Option 1

Mastery Fill

最贴近你的想法：选中的掌握按钮放大成整张反馈层，其他三个选项淡出。反馈非常明确，适合主方案。

Floating Status

Option 3

Review Ribbon

建议优先级： 我建议先落地 Option 1，因为它最符合你“放大填充、隐去其他选项”的想法，而且学生能立刻明白这一步是在记录掌握情况。Option 2 更适合后续做“快速复习模式”，Option 3 可以留给 session 完成或阶段总结时使用。
